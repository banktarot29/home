นโยบายความเป็นส่วนตัว

อัปเดตล่าสุด: มิถุนายน 2569

ซินแสแบงค์ใช้ข้อมูลที่คุณส่งมาเพื่อวิเคราะห์ฮวงจุ้ยเบื้องต้นเท่านั้น ระบบนี้ไม่บันทึกรูปภาพบ้านไว้ในฐานข้อมูลของเรา และไม่มีช่องฝาก LINE ID บนหน้าเว็บแล้ว หากต้องการคุยต่อ คุณต้องกดปุ่มไปทัก LINE ด้วยตัวเอง

ข้อมูลที่ใช้ในการวิเคราะห์

เมื่อคุณใช้งาน ระบบจะใช้ข้อมูลต่อไปนี้เพื่อสร้างผลวิเคราะห์:

  • ภาพหน้าบ้านที่คุณอัปโหลด โดยส่งต่อเพื่อประมวลผลและไม่เก็บไว้ในฐานข้อมูลของเรา
  • ข้อมูลที่คุณเลือก เช่น ประเภทบ้าน ทิศ สภาพแวดล้อม ปัญหาที่พบ การใช้งาน และระยะเวลาที่อยู่
  • IP Address เพื่อจำกัดการส่งคำขอซ้ำในช่วงเวลาสั้น ๆ

ข้อมูลที่ไม่เก็บ

  • ไม่เก็บรูปภาพบ้านไว้ในฐานข้อมูลของเรา
  • ไม่ขอหรือเก็บเบอร์โทรศัพท์
  • ไม่เก็บ LINE ID ผ่านฟอร์มบนหน้าเว็บ
  • ไม่ขายหรือส่งต่อข้อมูลผู้ใช้เพื่อการตลาด

การประมวลผลภาพ

ภาพและข้อมูลบ้านจะถูกส่งไปยังบริการประมวลผลภายนอกเพื่ออ่านภาพและสร้างผลวิเคราะห์ ข้อมูลที่ส่งผ่าน API ไม่ได้ถูกใช้เพื่อฝึกโมเดลตามนโยบายของผู้ให้บริการ API

Google Analytics

เราใช้ Google Analytics 4 เพื่อดูภาพรวมการใช้งาน เช่น จำนวนผู้เริ่มกรอก จำนวนผู้แนบรูป จำนวนการวิเคราะห์สำเร็จ และจำนวนการกดปุ่ม LINE ข้อมูลนี้ใช้เพื่อปรับปรุงหน้าเว็บ และไม่ได้เชื่อมกับตัวตนของคุณโดยตรง

การติดต่อผ่าน LINE

หากคุณกดปุ่ม LINE ระบบจะพาคุณออกไปยัง LINE Official Account ของซินแสแบงค์ การพูดคุยหลังจากนั้นจะอยู่ภายใต้ระบบและนโยบายของ LINE

ความปลอดภัย

การรับส่งข้อมูลใช้ HTTPS ภาพที่แนบจะถูกย่อขนาดในเบราว์เซอร์ก่อนส่ง เพื่อให้ประมวลผลได้เร็วและลดข้อมูลที่ไม่จำเป็น

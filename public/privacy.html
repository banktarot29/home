นโยบายความเป็นส่วนตัว

อัปเดตล่าสุด: มิถุนายน 2568

ซินแสแบงค์ให้ความสำคัญกับความเป็นส่วนตัวของคุณ ข้อมูลที่ส่งมาจะถูกใช้เพื่อวิเคราะห์ฮวงจุ้ยเท่านั้น ไม่มีการเก็บข้อมูลส่วนบุคคลหรือภาพบ้านของคุณไว้ในระบบ

ข้อมูลที่รับไว้ชั่วคราว

เมื่อคุณใช้งาน Fengshui Preview ระบบจะรับข้อมูลต่อไปนี้เพื่อวิเคราะห์เท่านั้น:

  • ภาพหน้าบ้านและภาพภายใน (ถ้ามี) — ส่งตรงไปยัง Anthropic API เพื่อวิเคราะห์ ไม่มีการบันทึกในฐานข้อมูลของเรา
  • ข้อมูลที่คุณเลือก เช่น ประเภทบ้าน ทิศ สภาพแวดล้อม ปัญหาที่พบ — ใช้สร้าง prompt วิเคราะห์เท่านั้น
  • IP Address — ใช้จำกัดการส่งคำขอซ้ำในเวลาสั้น (Rate Limiting) เก็บไว้ชั่วคราวใน memory ไม่เกิน 10 นาที

ข้อมูลที่ไม่เก็บ

  • ภาพบ้านของคุณ — ไม่มีการบันทึกหรือเก็บไว้ในเซิร์ฟเวอร์
  • ชื่อ ที่อยู่ หมายเลขโทรศัพท์ หรือข้อมูลส่วนตัวอื่นใด
  • ผลการวิเคราะห์ — ไม่มีการบันทึกผลเชื่อมกับตัวคุณ
  • Cookie ที่ใช้ติดตามพฤติกรรม

Google Analytics (GA4)

เราใช้ Google Analytics 4 เพื่อวัดจำนวนผู้ใช้และพฤติกรรมการใช้งานโดยรวม (เช่น จำนวนคนที่วิเคราะห์สำเร็จ อัตราการกด LINE) ข้อมูลนี้ไม่มีการเชื่อมกับตัวตนของคุณ และอยู่ภายใต้ นโยบายของ Google

Anthropic API

ภาพและข้อมูลบ้านจะถูกส่งไปยัง Anthropic เพื่อประมวลผล AI ข้อมูลอยู่ภายใต้ นโยบายความเป็นส่วนตัวของ Anthropic โดย Anthropic ไม่ใช้ข้อมูลที่ส่งผ่าน API เพื่อ train โมเดล

ความปลอดภัย

การรับ-ส่งข้อมูลทั้งหมดเข้ารหัสด้วย HTTPS ภาพที่แนบจะถูก compress และส่งเป็น base64 ผ่าน API โดยตรง ไม่มีการเขียนลงดิสก์หรือ storage ใดๆ

สิทธิ์ของคุณ

เนื่องจากเราไม่เก็บข้อมูลส่วนบุคคล จึงไม่มีข้อมูลให้ลบหรือขอดู หากมีข้อสงสัยติดต่อได้ผ่านช่องทางด้านล่าง

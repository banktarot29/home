นโยบายความเป็นส่วนตัว

อัปเดตล่าสุด: มิถุนายน 2569

ซินแสแบงค์ให้ความสำคัญกับความเป็นส่วนตัวของคุณ ข้อมูลที่ส่งมาจะใช้เพื่อวิเคราะห์ฮวงจุ้ยเบื้องต้นเท่านั้น และจะไม่เก็บรูปภาพบ้านของคุณไว้ในระบบของเรา

ข้อมูลที่ใช้ในการวิเคราะห์

เมื่อคุณใช้งาน Fengshui Preview ระบบจะใช้ข้อมูลต่อไปนี้เพื่อสร้างผลวิเคราะห์:

  • ภาพหน้าบ้านและภาพภายใน ถ้ามี โดยส่งไปยัง Anthropic API เพื่อวิเคราะห์ ไม่บันทึกไว้ในระบบของเรา
  • ข้อมูลที่คุณเลือก เช่น ประเภทบ้าน ทิศ สภาพแวดล้อม ปัญหาที่พบ การใช้งาน และระยะเวลาที่อยู่
  • IP Address เพื่อจำกัดการส่งคำขอซ้ำในช่วงเวลาสั้น ๆ

ระบบฝากผลไว้ติดต่อกลับ

หากคุณเลือกฝาก LINE ไว้ ระบบจะบันทึกเฉพาะข้อมูลที่จำเป็นสำหรับติดตามผล เช่น LINE ID หรือชื่อ LINE, สรุปผลวิเคราะห์, คะแนน และข้อมูลบ้านที่เลือกไว้ โดยไม่เก็บรูปภาพที่แนบ

ถ้าเปิดผ่าน LINE LIFF ระบบอาจอ่านชื่อ LINE และ LINE user id เพื่อช่วยกรอกข้อมูลให้ง่ายขึ้น ข้อมูลนี้ใช้เพื่อการติดต่อกลับเท่านั้น

ข้อมูลที่ไม่เก็บ

  • ไม่เก็บรูปภาพบ้านไว้ในฐานข้อมูลของเรา
  • ไม่ขอหรือเก็บเบอร์โทรศัพท์
  • ไม่ขายหรือส่งต่อข้อมูลผู้ใช้ให้บุคคลภายนอกเพื่อการตลาด
  • ไม่ใช้ cookie เพื่อติดตามตัวตนรายบุคคล

Google Analytics

เราใช้ Google Analytics 4 เพื่อดูภาพรวมการใช้งาน เช่น จำนวนผู้ใช้ จำนวนครั้งที่วิเคราะห์สำเร็จ และการกดปุ่ม LINE ข้อมูลนี้ใช้เพื่อปรับปรุงหน้าเว็บและไม่ได้เชื่อมกับตัวตนของคุณโดยตรง อ่านเพิ่มเติมได้ที่ นโยบายของ Google

Anthropic API

ภาพและข้อมูลบ้านจะถูกส่งไปยัง Anthropic เพื่อประมวลผล AI ข้อมูลอยู่ภายใต้ นโยบายความเป็นส่วนตัวของ Anthropic โดยข้อมูลที่ส่งผ่าน API จะไม่ถูกใช้เพื่อ train โมเดล

ความปลอดภัย

การรับส่งข้อมูลทั้งหมดใช้ HTTPS ภาพที่แนบจะถูกย่อขนาดก่อนส่ง และส่งผ่าน API เพื่อประมวลผลเท่านั้น

สิทธิ์ของคุณ

หากต้องการให้ลบข้อมูลที่ฝากไว้ หรือต้องการสอบถามเรื่องความเป็นส่วนตัว สามารถติดต่อผ่าน LINE ด้านล่างได้
